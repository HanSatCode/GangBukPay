# 소개
> 축제 부스 체험을 더욱 즐겁고 재미있게

**강북페이**는 강북고 내 동아리 부스 참여 독려를 위한 리워드 웹페이지입니다.
부스 참여를 통해 쿠폰 코드를 얻어 포인트로 교환할 수도 있고, 룰렛을 통해 상품으로 교환할 수도 있죠.
그러나, 개발할 당시에는 코로나19로 인해 사용은 못 하게 되었지만, 언제나 필요할 때 쓰일 수 있도록 소스코드를 개발하고 있어요.
